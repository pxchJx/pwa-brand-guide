Steps 1 – 4

What to change

1

Change the app name

One line in JavaScript

script — top of <script> block 
// Change "YOURNEWBRAND" to your brand name
window.APP_NAME = window.APP_NAME || "YOURNEWBRAND";

Also update the page title

Find <title> in <head> and change it to your brand name.

2

Edit the :root CSS variables

The only section you change for colors — find :root { } at the top of the CSS

Critical

Do not touch

These sections were carefully debugged and must not be edited

Editing any of these will break modal positioning, SPA re-navigation, smart banner sync, or the Webflow tab engine. Do not touch them even if you think you know what they do.

JavaScript functions — never edit

measureTopOffset()
measureBottomOffset()
startModalSync()
stopModalSync()
showModal()
hideModal()
renderInstructionCards()
initTabs()
bindEvents()
checkNotificationPermission()
checkPWAStatus()

pushNotification()
handleInstallPWA()
handlePushButtonClick()

handleAppBrowserRedirect()
InstructionCard class

else branch (SPA re-nav handler)
_pwaRunChecks / _pwaReInit hooks
MODAL_DATA keys & structure

CSS sections — never edit

/* RESET / BASE */ section
.body and .page rules
benefit-grid column layout
All display:flex/grid rules
/* Webflow tab engine */ block
.w-tabs / .w-tab-* / .w--tab-active
/* RESPONSIVE */ media queries
.modal-overlay base rules
.modal-overlay.active rule
--container: 495px value
